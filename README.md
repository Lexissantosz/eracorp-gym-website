eracorp-gym-website
Modern responsive landing page for ERACORP Gym. Built with HTML, CSS and JavaScript, featuring sections for plans, benefits, testimonials, gym structure, units and a functional contact form. Designed with a clean purple identity and focused on user experience and conversions.

ERACORP Gym Website

Landing page moderna desenvolvida para a academia **ERACORP**, com foco em apresentação profissional da academia, captação de novos alunos e experiência visual moderna.

O site possui identidade visual baseada em tons de **roxo**, com design minimalista inspirado em grandes redes de academia.

Objetivo do projeto

Criar uma presença digital profissional para a academia **ERACORP**, permitindo que novos alunos conheçam a academia, seus planos, estrutura e entrem em contato facilmente.

O projeto também serve como estudo e prática de desenvolvimento frontend focado em landing pages profissionais.
